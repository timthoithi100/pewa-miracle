# Pewa Miracle Travel & Tours Ltd

**Pewa Miracle Travel & Tours Ltd** is a high-end, responsive landing page designed for a fully licensed travel agency based in Nairobi, Kenya. The website emphasizes "Seamless Experiences," showcasing services ranging from luxury safaris to international corporate travel.

---

## Features

* **Modern Premium Aesthetic**: Utilizes a sophisticated "Jungle Green" and "Cream" color palette to reflect nature and luxury.
* **Fully Responsive**: Optimized for desktop, tablet, and mobile devices using CSS Grid and Flexbox.
* **Interactive UI**: Includes smooth scroll behavior, hover-activated destination cards, and entry animations.
* **Comprehensive Service Showcase**: Dedicated sections for:
    * Flight, Train, and Hotel Bookings
    * Safari & Wildlife Tours
    * Visa & Travel Advisory
    * Adventure & Hiking
* **Dynamic Destination Gallery**: A multi-layout grid featuring iconic locations like the Holy Land, Kenya, Europe, and the Americas.
* **Direct Communication**: Integrated CTA (Call to Action) band with one-click links for Phone, WhatsApp, and Email.

---

## Design System

| Element | Specification |
| :--- | :--- |
| **Primary Colors** | Deep Green (`#0d3b1e`), Bright Green (`#2db84b`), Gold (`#c9a84c`) |
| **Typography** | *Cormorant Garamond* (Serif) & *DM Sans* (Sans-serif) |
| **Icons** | Emojis used for lightweight visual cues |
| **Images** | High-resolution curated travel photography via Unsplash |

---

## Built With

* **HTML5**: Semantic structure for SEO and accessibility.
* **CSS3**: Custom properties (Variables), Keyframe animations, Flexbox, and CSS Grid.
* **Google Fonts**: Integrated via `<link>` for typography.

## Getting Started

To view the website locally:

1. **Clone the repository**:
   ```bash
   git clone [https://github.com/your-username/pewa-miracle.git](https://github.com/your-username/pewa-miracle.git)
   ```
2. **Navigate to the directory**:
   ```bash
   cd pewa-miracle-travel
   ```
3. **Open the file**:
   Simply open `index.html` in your preferred web browser.
